Projetos

Analise de comercio exterior Estado de SP

Projeto de entrega de uma plataforma de analise e visualização de dados do comercio exterior nos ultimos 10 anos

Descrição

Desenvolvemos uma Plataforma web para visualização dos dados do comercio exterior do estado de São Paulo, exibindo informações sobre exportação e importação de cada municipio

Desafio

Ver mais...

Nesse projeto fiquei responsável pela limpeza e estruturação de um dataframe com 10 milhões de linhas, otimizando-o para uso no Google Colab com baixo consumo de hardware. Realizei a análise de dados e criação de gráficos conforme os requisitos do cliente. Desenvolvi o backend em Python com Flask, integrando o sistema a um banco de dados MySQL e conectando o backend ao frontend para exibição dinâmica dos gráficos gerados a partir das consultas.
